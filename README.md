# IllySurveyAdmin

Bu proje, Illy Survey Admin sistemidir.

## Proje Yapısı

- `src/main/java/` - Java kaynak kodu
- `src/main/resources/` - Kaynak dosyaları (log4j2.xml vb.)
- `src/main/webapp/` - Web uygulaması dosyaları

## Kullanılan Teknolojiler

- Java
- Apache Tomcat
- Log4j2

## Kurulum ve Çalıştırma

1. Projeyi klonlayın:
```bash
git clone <repository-url>
```

2. Eclipse'de projeyi açın

3. Tomcat sunucusu üzerinde çalıştırın
